## pairing skill
Pairs the mycroft device to home.mycroft.ai

## Description 
Pairs the mycroft device to home.mycroft.ai and gives you a code used to add the device at home.mycroft.ai

## Examples 
* en-us
 * "pair my device"
 * "register my device"
 * "pair my unit"
* es-es
 * "emparejar mi dispositivo"
 * "registrar mi dispositivo"
 * "emparejar mi unidad"
